In the current use, an LED panel is velcroed to a foam head with soldered wires connecting into an Arduino Uno Mini. 
In addition, a potentiometer with soldered wires is situated atop a box with labels for each of the 3 modeled conscious/wakeful 
states (Anesthetized, Wake, Seizure). This is connected to the Arduino. When the potentiometer reads a value in the range associated with 
each state, the Arduino causes the light panel to execute the corresponding sequence. The LED activity mimics the behavior of EEG neural
avalanche criticality and the functional connectivity of EEG electrodes in that state. 

Code was written in Visual Studio Code.
